Expression format

All input math expression should be expressed in linear infix notation. For example:

  • 1 + 2 x^2
  • a sin(wx + f)

Literal numbers are expressed in the same form as float in C language:

  • 3.14159265
  • 1.23e-8
Following math functions are supported:
  • sqrt: square root
  • log: common logarithm
  • ln: nature logarithm
  • exp: exponential function
  • sin: sine function
  • cos: cosine function
  • tan: tangent function
